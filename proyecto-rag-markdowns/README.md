# RAG sobre capital-trabajo y wage markdowns

Este proyecto construye un sistema Retrieval-Augmented Generation (RAG) para apoyar una investigacion economica sobre automatizacion, intensidad de capital y poder de negociacion laboral.

La entrega del curso se enfoca en el sistema RAG: documentos reales, chunking, embeddings, base vectorial, recuperacion, generacion, evaluacion y trazabilidad. La pregunta empirica completa queda planteada como extension para un paper futuro.

## Estructura

- `data/raw`: documentos originales.
- `data/processed`: textos extraidos, chunks y datos procesados.
- `notebooks`: notebook principal del proyecto.
- `src`: codigo auxiliar del pipeline RAG.
- `report`: reporte final en PDF y fuentes editables.
- `video`: guion y estructura de la sustentacion.
- `docs`: notas metodologicas y bibliografia.
